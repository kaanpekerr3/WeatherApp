# 🌤 WeatherApp

ASP.NET Core Razor Pages kullanılarak geliştirilmiş anlık hava durumu uygulaması.

## 🚀 Özellikler

- 🌍 Şehir ve ilçe arama
- 🌡️ Anlık sıcaklık
- 🥵 Hissedilen sıcaklık
- 💧 Nem oranı
- 💨 Rüzgar hızı
- ☁️ Hava durumu açıklaması
- ⏰ Güncellenme zamanı
- 📱 Responsive tasarım

## 🛠️ Teknolojiler

- ASP.NET Core Razor Pages
- C#
- Open-Meteo API
- HTML
- CSS
- Bootstrap
